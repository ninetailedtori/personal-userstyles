a cozy collection of custom styles for the web, made with
[stylus](https://github.com/openstyles/stylus) ✨

---

## what is this?

this repo contains userstyles i've made to customize the look and feel of
websites i spend time on. with the stylus extension, you can install custom CSS
that overrides a website's default styling—perfect for darkening bright sites,
adjusting fonts, hiding annoying elements, or just making things a little more
_you_.

---

## install stylus

grab the extension/source code here: https://github.com/openstyles/stylus

---

## how to use

(´∇｀) once stylus is installed:

1. visit one of the sites covered by these styles
2. click the stylus icon in your browser toolbar
3. click the plus icon to install a new style
4. paste the code from any file in this repo into the userstyles editor!

each style file includes a comment at the top explaining what it customizes and
which site(s) it works with.

---

## the collection

you'll find styles here for various websites. feel free to mix and match, modify
for your own taste, or use them as inspiration for creating your own styles
ᐡ•ﻌ•ᐡ

---

## tips

- **tweak anything:** userstyles are just css - edit them to match your vibe
  exactly
- **toggle easily:** use stylus to enable/disable styles on the fly
- **combine styles:** many styles work great together
- **check compatibility:** some styles might need tweaks if website layouts
  update
